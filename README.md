# Personal Expense Splitter App

An Android application that helps users split shared expenses between multiple people and automatically calculate balances.

## Features

- Add shared expenses
- Assign participants to each expense
- Automatically calculate how much each person owes
- Track balances between users
- Simple and intuitive Android interface

## Technologies Used

- Java
- Android Studio
- XML Layouts
- Android SDK

## How It Works

Users can enter an expense and select the participants involved.  
The application calculates how the cost should be divided and tracks the balance between users.
