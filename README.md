# 💰 Personal Expense Splitter App

A simple Android application that helps users split shared expenses between multiple people and automatically calculate balances.  
The app is designed to make group expense tracking easier, transparent, and organized.

---

## ✨ Features

- Add shared expenses with custom amounts
- Assign participants to each expense
- Automatically calculate how much each person owes or is owed
- Track balances between multiple users
- Clean and simple user interface
- Lightweight and easy-to-use design

---

## ⭐ Highlights

- Built with a structured Android architecture using Activities and XML layouts
- Real-time calculation of balances between participants
- Beginner-friendly project with scalable design for future improvements
- Focus on logic, data handling, and user interaction

---

## 🛠️ Technologies Used

- Java
- Android Studio
- XML Layouts
- Android SDK

---

## 📁 Project Structure

- `activities/` → App screens and logic
- `models/` → Data structures for expenses and users
- `adapters/` → (if used) List handling components
- `res/layout/` → XML UI layouts
- `res/drawable/` → Icons and UI assets

---

## 🚀 How It Works

1. User adds an expense amount
2. Participants are selected for that expense
3. The app divides the cost among selected users
4. Balances are automatically calculated and updated
5. Users can track who owes whom at any time
